### 1. 简介
    
    针对中文恶意网页检测的数据稀缺，难以获取的问题。构建并发布了中文恶意网页检测数据集。
    每一个样本包含网页URL、HTML网页文件、JavaScript代码文件。

### 2. 数据规模
    
    包含521个中文恶意网页。恶意网页的种类涉及博彩类、钓鱼类（冒充正常页面）、色情类、违法交易类等。
