# Chinese_Malicious_Web_Pages_Dataset_And_Detection
中文恶意网页检测数据集与检测方法

## 中文恶意网页检测数据集与检测方法



### 1. 简介
    
    针对中文恶意网页检测的数据集稀缺的问题。构建并发布了包含网页URL、HTML网页文件、JavaScript代码文件的中文恶意网页检测数据集。

### 2. 数据规模
    
    包含521个中文恶意网页。恶意网页的种类涉及博彩类、钓鱼类（冒充正常页面）、色情类、违法交易类等。

    中文恶意网页列表（含URL）的地址：见项目的data文件。
